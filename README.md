# Cisco Networking Labs

Hands-on Cisco Packet Tracer networking labs focused on network configuration, troubleshooting, routing, switching, IPv4/IPv6 addressing, wireless networking, and network services.

## Overview

This repository contains practical Cisco Packet Tracer (`.pka` / `.pksz`) networking labs completed as part of Cisco Networking Academy coursework and self-practice.

The labs simulate real-world networking scenarios including:
- Router and switch configuration
- IPv4 and IPv6 addressing
- Network troubleshooting
- Routing and switching
- Wireless networking
- NAT and DHCP
- SSH and Telnet remote access
- TCP/UDP communication
- LAN/WAN connectivity
- Network services and diagnostics
